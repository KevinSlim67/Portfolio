Project Overview

This is a website created for a fictional company that sells electronic smart devices. This was my first MERN Stack application. Meaning it was made using MongoDB, Express.js, React.js and Node.js. It made me much more comfortable with frontend, and it was also my first introduction to backend.

The website is fully responsive, and works on all devices.
